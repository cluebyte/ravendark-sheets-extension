# Privacy Policy for RavenDark Sheets to Roll20

**Last updated:** [Date]

## Summary

**RavenDark Sheets to Roll20** does not collect, store, or transmit any personal or usage data. All behavior stays inside your browser.

## What this extension does

- On RavenDark character sheet pages, it adds “Roll” buttons next to rollable values.
- When you click a roll button, it sends the corresponding roll formula (e.g. `1d20+5`) to an open Roll20 game tab’s chat.

## Data handling

- **No accounts.** The extension does not use accounts or log you in anywhere.
- **No analytics.** No usage or analytics data is sent to any server.
- **No storage of your data.** It does not save character data, roll results, or any other information to disk or to a server.
- **Permissions.** It uses the `tabs` and `scripting` permissions and access only to:
  - `https://ravendark-sheets.vercel.app/characters/*` (to add Roll buttons), and
  - `https://app.roll20.net/*` (to send roll commands to the Roll20 chat).  
    These are used only for the features described above.

## Contact

If you have questions about this policy, you can open an issue in the extension’s repository or contact the developer via the Chrome Web Store listing.
