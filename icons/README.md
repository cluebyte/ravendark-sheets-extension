# Extension icons

Add PNG icons here for Chrome Web Store and the extension UI:

| File       | Size    | Use |
|------------|---------|-----|
| icon16.png | 16×16   | Favicon for extension pages |
| icon48.png | 48×48   | Extensions management page |
| icon128.png| 128×128 | **Required** – Store listing & install |

Use the same design at each size. PNG only (no WebP or SVG in manifest).
